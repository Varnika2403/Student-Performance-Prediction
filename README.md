# Student-Performance-Prediction
Machine Learning project to predict student performance using regression models and data analysis techniques.
## Project Overview

This project predicts student performance based on various academic and lifestyle factors using Machine Learning techniques.

## Features Used

* Hours Studied
* Previous Scores
* Extracurricular Activities
* Sleep Hours
* Sample Question Papers Practiced

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Machine Learning Models

* Linear Regression
* Ridge Regression
* Lasso Regression
* ElasticNet
* K-Nearest Neighbors (KNN)
* Decision Tree Regressor
* Random Forest Regressor

## Project Workflow

1. Data Loading and Exploration
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Performance Comparison

## Evaluation Metrics

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score

## Outcome

Successfully built and evaluated multiple machine learning models to predict student performance and compared their effectiveness using standard regression metrics.
